# Chronos - 时间管理大师

一款面向个人用户的时间管理工具，帮助用户追踪、分析和优化时间使用情况，提高时间管理效率。

## 产品特点

- 时间可视化：直观展示时间分配情况
- 数据分析：提供时间使用的统计和洞察
- 行为优化：基于数据提供时间管理建议

## 目标用户

- 职场人士：需要平衡工作与生活的白领
- 学生群体：需要合理安排学习与课外活动时间的学生
- 自由职业者：需要灵活管理时间的个体工作者

## 技术架构

- 前端：uni-app + uView UI + ECharts
- 后端：uniCloud + 云数据库 + Node.js
